## Implementation Details

### 1. ✅ Backend Image Processing

**Library Used:** `sharp` (already installed and configured)

**Location:** `/server/helpers/watermark.js`

Two watermark functions implemented:

#### Text Watermark (`applyWatermark`)
- Generates SVG text overlay
- Configurable text, position, opacity, font size
- Default: "Royal Motors" in white with shadow
- Responsive font sizing based on image dimensions

#### Logo Watermark (`applyLogoWatermark`)
- Loads logo from file (SVG or PNG)
- Resizes logo relative to image size (default 15% of width)
- Configurable position, opacity, margin
- Fallback to text watermark if logo fails

### 2. ✅ Integration with Upload Flow

**Location:** `/server/controllers/vehiclesController.js`

Watermarking is integrated into both vehicle creation and update:

```javascript
// Lines 176-198: Apply watermark before R2 upload
const enableWatermark = process.env.ENABLE_WATERMARK === 'true';
if (enableWatermark) {
  const logoPath = process.env.WATERMARK_LOGO_PATH ||
    path.join(__dirname, '../../static/logo-watermark.svg');
  const useLogoWatermark = process.env.USE_LOGO_WATERMARK === 'true';

  if (useLogoWatermark) {
    imageBuffer = await applyLogoWatermark(imageBuffer, logoPath, {
      position: 'southeast',
      scale: parseFloat(process.env.WATERMARK_SCALE) || 0.15,
      opacity: parseFloat(process.env.WATERMARK_OPACITY) || 0.7,
      margin: parseInt(process.env.WATERMARK_MARGIN) || 20,
    });
  } else {
    imageBuffer = await applyWatermark(imageBuffer, {
      text: process.env.WATERMARK_TEXT || 'Royal Motors',
      position: 'bottom-right',
      opacity: parseFloat(process.env.WATERMARK_OPACITY) || 0.6,
      margin: 20,
    });
  }
}
```

## Configuration Options

### Enable/Disable Watermarking

Set `ENABLE_WATERMARK="false"` to disable all watermarking.

### Switch Between Logo and Text

- Logo watermark: `USE_LOGO_WATERMARK="true"` (current)
- Text watermark: `USE_LOGO_WATERMARK="false"`

### Adjust Logo Appearance

- **Size:** `WATERMARK_SCALE` (0.15 = 15% of image width)
- **Opacity:** `WATERMARK_OPACITY` (0.7 = 70% opaque)
- **Margin:** `WATERMARK_MARGIN` (20 = 20px from edges)

### Change Logo File

Update `WATERMARK_LOGO_PATH` to point to a different logo file (SVG or PNG).

### Change Text Watermark

Update `WATERMARK_TEXT="Your Text Here"` for text mode.
