## 🔧 Technical Implementation

### Watermarking Flow

```
User uploads image
       ↓
Multer stores in memory buffer
       ↓
Check ENABLE_WATERMARK=true
       ↓
Load logo SVG
       ↓
Resize logo to 15% of image width
       ↓
Composite at bottom-right with 70% opacity
       ↓
Upload watermarked image to R2
       ↓
Save URL to database
       ↓
Display in UI (with watermark)
```

### Error Handling

```
Logo watermark attempt
       ↓
   [Failed?]
       ↓ Yes
Text watermark fallback
       ↓
   [Failed?]
       ↓ Yes
Return original image (no watermark)
       ↓
Log error, continue upload
```

### Performance

- Processing time: ~50-200ms per image
- No temporary files (in-memory processing)
- Original quality preserved
- Aspect ratio maintained
- Supports images up to 5MB

## 🛡️ Security & Protection

### Public Page Protection
All images are watermarked on upload, including those shown on:
- Public tracking pages (`/track/:vin`)
- Car detail pages
- Car list tables
- Export/download features

This protects vehicle images from unauthorized use while allowing customers to track their vehicles.

### Storage Strategy
- **Watermarked images only** are stored in R2
- No separate original copies stored
- Reduces storage costs
- Ensures all images (old/new) use watermark
- Can't accidentally serve non-watermarked images

## 📝 Notes

### Why Logo Instead of Text?
- More professional appearance
- Brand recognition
- Better visual integration
- Scalable to any image size
- Maintains quality at all resolutions

### Why 70% Opacity?
- Visible enough to protect images
- Transparent enough to not obstruct vehicle details
- Balanced between branding and usability
- Industry standard for watermarks

### Why Bottom-Right?
- Standard watermark position
- Doesn't cover main subject (car front/side)
- Easy for viewers to locate
- Consistent across all images
- Professional convention

### Why 15% Scale?
- Large enough to be clearly visible
- Small enough to not dominate image
- Responsive to different image sizes
- Maintains readability on small images
- Professional appearance on large images
